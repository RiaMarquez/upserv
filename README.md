# UpServ

Service website for UpServ.

## Project Structure

```
upserv/
├── index.html          # Entry point
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # Main script
├── assets/
│   ├── images/         # Image assets
│   └── fonts/          # Custom fonts
├── pages/              # Additional pages
├── .gitignore
└── README.md
```

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/RiaMarquez/upserv.git
   ```
2. Open `index.html` in your browser to preview.
3. Edit files and add pages as needed.

## Contributing

- Add new pages in the `pages/` directory.
- Place images in `assets/images/` and fonts in `assets/fonts/`.
- Keep shared styles in `css/styles.css`.
- Keep shared scripts in `js/main.js`.
